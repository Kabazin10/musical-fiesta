# Musical Fiesta

## Project Overview
Musical Fiesta is a comprehensive musical festival management project that includes both a frontend and backend application. This project aims to provide a platform for organizers to manage events, artists, and ticket sales efficiently.

## Table of Contents
- [Backend](#backend)
- [Frontend](#frontend)
- [Docker](#docker)
- [Configuration Files](#configuration-files)

## Backend
The backend is built using Express.js and Node.js. It manages the server-side logic, APIs, and database connections.

### Features
- User authentication
- CRUD operations for events
- Artist management
- Ticket sales

### Technologies
- Node.js
- Express.js
- MongoDB

## Frontend
The frontend is built using React. It allows users to interact with the application and provides a user-friendly interface.

### Features
- Responsive design
- Event browsing
- User registration and login
- Ticket purchasing

### Technologies
- React
- Redux
- CSS frameworks (e.g., Bootstrap)

## Docker
Docker files are provided to facilitate containerization of both frontend and backend applications for easy deployment.

## Configuration Files
Configuration files are set up for both development and production environments. They manage environment variables, database connections, and API keys.

## Installation
- Clone the repository
- Install dependencies for both frontend and backend
- Start the development servers

## Usage
- Access the frontend application at `http://localhost:3000`
- Access the backend API at `http://localhost:5000`
